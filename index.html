Course

About the course

An 8-week live cohort on no-code AI tools, automations, and research communication. Features weekly workshops, help sessions, community support, personalized feedback, a capstone project, and a shareable certificate.

Module 1 · Foundations

How LLMs are trained · Generating Markdown/LaTeX/HTML · Building simple research websites.

Module 2 · AI-Automated Data Analysis

Code Interpreter & Julius · Automated dashboards · File conversion · Data viz & reports.

Module 3 · GenAI for Research/Text

Literature reviews with Elicit/LiteRev · Text classification · Sentiment · Knowledge maps.

Module 4 · Custom GPTs & Chatbots

LLM chat architecture · RAG · Knowledge bases · Interactive assistants.

Module 5 · AI for Automation

Agent-based flows · Email response automations · Browser control agents.

Module 6 · Communication & Outreach

AI-generated summaries · Scientific figures · Video abstracts · Visualization prompts.

Module 7 · Capstone & Showcase

Project coaching · Peer feedback · Final presentation & public Q&A.

Grading

Quizzes 15% · Assignments 35% · Participation 10% · Final project 40% (Passing ≥ 80%).

Bootcamp Features

Live workshops & help sessions · Private community · Lifetime tutorials · Affordable non-profit program.
